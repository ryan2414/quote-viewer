# Quote Viewer 개발 로드맵

매일의 명언과 성경 말씀으로 사용자의 하루에 영감을 더하는, SEO 친화적이고 공유하기 좋은 명언 뷰어 서비스.

## 개요

Quote Viewer는 위로와 영감이 필요한 일반 사용자를 위한 **매일의 명언/말씀 큐레이션 서비스**로 다음 기능을 제공합니다:

- **SEO 기반 콘텐츠 발견**: 명언별 정적 페이지와 구조화된 메타데이터로 검색 유입 극대화
- **소셜 공유 & 바이럴**: 카카오톡/X 공유와 명언 카드 이미지 저장으로 자연스러운 확산
- **습관 형성 경험**: 오늘의 명언, 연속 방문 스트릭, 푸시 알림으로 재방문 유도
- **고도화 & 수익화**: 사용자 계정, 개인화 추천, 다국어, 후원 연동

## 단계별 목표 (KPI)

## 기술 스택

- **Next.js 16.2.6** (App Router, SSG/SSR)
- **React 19.2.4** + **TypeScript**
- **Tailwind CSS 4**
- **Supabase** (인증 / DB) — `@supabase/ssr`, `@supabase/supabase-js`
- **@vercel/analytics** — 기본 분석
- **테스트**: Playwright MCP (E2E / API 연동 검증)

## 개발 워크플로우

1. **작업 계획**
   - 기존 코드베이스를 학습하고 현재 상태를 파악
   - 새로운 작업을 포함하도록 `ROADMAP.md` 업데이트
   - 우선순위 작업은 마지막 완료된 작업 다음에 삽입

2. **작업 생성**
   - 고수준 명세서, 관련 파일, 수락 기준, 구현 단계 포함
   - API/비즈니스 로직 작업 시 "## 테스트 체크리스트" 섹션 필수 포함 (Playwright MCP 테스트 시나리오 작성)
   - 새 작업은 빈 체크박스로 작성하고, 완료 시 ✅로 표시

3. **작업 구현**
   - 작업 파일의 명세서를 따라 기능 구현
   - API 연동 및 비즈니스 로직 구현 시 Playwright MCP로 테스트 수행 필수
   - 각 단계 완료 후 진행 상황 업데이트 및 추가 지시 대기

4. **로드맵 업데이트**
   - 로드맵에서 완료된 작업을 ✅로 표시
