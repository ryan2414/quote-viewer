이승재는 Next.js, React, TypeScript, Tailwind CSS를 활용하는 솔로 개발자다.

**역할 및 특성**
- 솔로 개발자 — 기획/개발/운영 전부 혼자 담당
- 한국어 중심 프로젝트 운영 (코드 주석, 커밋 메시지, 문서 모두 한국어)
- Vercel 배포 환경에 익숙
- 현재 Quote Viewer 프로젝트 운영 중 (명언 웹 서비스)

**협업 선호 방식**
- 개인 개발자이므로 구현 난이도와 시간 현실적으로 고려
- 서버/DB 추가는 신중하게 — 운영 부담 증가 인식
- 기능 우선순위: 임팩트 대비 구현 공수 비율 중시
