# Quote Viewer 개발 로드맵 V2

주제별 필터링으로 원하는 카테고리의 명언을 탐색하는 기능 추가

## 개요

Phase 5는 기존 52개 명언 데이터에 이미 정의된 카테고리 필드를 UI에 노출시켜,
사용자가 **지혜 / 동기부여 / 인생 / 용기 / 사랑** 주제별로 명언을 탐색할 수 있게 한다.

- **카테고리 배지 표시 (F011)**: 명언 카드와 상세 뷰에 카테고리 레이블을 배지로 표시
- **카테고리 필터 탭 (F012)**: 명언 목록 페이지에 카테고리별 탭 추가 및 필터링

> **참고**: 카테고리 데이터(wisdom/motivation/life/courage/love)는 `data/quotes.ts`에 이미 존재한다.
> 이 Phase는 해당 데이터를 UI에 연결하는 작업이다. 서버/DB 변경 없음.

2. **작업 생성**
   - 기존 코드베이스를 학습하고 현재 상태를 파악
   - 고수준 명세서, 관련 파일, 수락 기준, 구현 단계 포함
   - 비즈니스 로직/영속성 작업 시 "## 테스트 체크리스트" 섹션 필수 포함 (Playwright MCP 테스트 시나리오 작성)

## 개발 단계

### Phase 5: 주제별 명언 기능

> 목표: 카테고리 메타데이터를 정의하고, 명언 카드/상세 뷰에 배지를 추가하며,
> 명언 목록 페이지에 카테고리 필터 탭을 구현한다.

- **Task 014: 카테고리 메타데이터 및 유틸리티 확장** - 우선순위
  - `data/categories.ts` 신규 — 카테고리 id, 한국어 label, Tailwind 색상 클래스 정의
    ```
    wisdom     → 지혜      (blue)
    motivation → 동기부여  (green)
    life       → 인생      (violet)
    courage    → 용기      (orange)
    love       → 사랑      (rose)
    ```
  - `types/quote.ts` — `category` 필드를 `string`에서 `CategoryId` 유니온 타입으로 교체
  - `data/quotes.ts` — `getQuotesByCategory(category: CategoryId)` 유틸 추가
  - **수락 기준**: `getQuotesByCategory('wisdom')` 호출 시 wisdom 명언만 반환, `npm run build` 타입 오류 없음

- **Task 015: QuoteCard / QuoteDisplay 카테고리 배지 추가**
  - `components/QuoteCard.tsx` — 저자명 옆에 카테고리 배지 추가 (카테고리별 색상)
  - `components/QuoteDisplay.tsx` — 저자명 아래에 카테고리 배지 표시
  - `data/categories.ts`의 메타데이터를 활용해 label/색상 조회
  - **수락 기준**: 명언 카드와 상세 뷰에 올바른 카테고리 레이블이 해당 색상으로 표시됨

  #### 테스트 체크리스트
  - 명언 카드에 카테고리 배지가 렌더링되는지 검증
  - 홈 페이지 상세 뷰에 카테고리 배지가 표시되는지 검증
  - 카테고리 없는 명언(category 미정의)에서 배지가 나타나지 않는지 검증

- **Task 016: 명언 목록 카테고리 필터 탭 구현**
  - `app/quotes/page.tsx` — `TabType` 확장
    ```typescript
    type TabType = 'all' | 'wisdom' | 'motivation' | 'life' | 'courage' | 'love' | 'favorites';
    ```
  - 탭 바에 카테고리 탭 5개 추가 (각 탭에 해당 카테고리 명언 수 배지 표시)
  - `getQuotesByCategory()`를 활용해 선택된 탭에 맞는 명언 카드 그리드 렌더링
  - 탭이 많을 경우 모바일에서 가로 스크롤 처리
  - **수락 기준**: 탭 전환 시 해당 카테고리 명언만 표시되고, 명언 수 배지가 실제 데이터와 일치

  #### 테스트 체크리스트
  - 각 카테고리 탭 클릭 시 해당 카테고리 명언만 필터링되는지 검증
  - 탭의 명언 수 배지가 실제 명언 수와 일치하는지 검증
  - 카테고리 탭에서 카드 클릭 시 홈으로 이동하며 해당 명언이 표시되는지 검증
  - 카테고리 탭에서 즐겨찾기 토글이 정상 동작하는지 검증
  - 모바일(375px)에서 탭 가로 스크롤이 동작하는지 검증

- **Task 017: 카테고리 기능 E2E 테스트 및 배포 검증**
  - Playwright MCP로 카테고리 필터 전체 사용자 여정 검증
  - `npm run lint` 및 `npm run build` 무오류 확인
  - Vercel 배포 후 프로덕션 동작 점검
  - **수락 기준**: 전체 카테고리 필터 플로우 끊김 없음, 빌드/린트 무오류

  #### 테스트 체크리스트
  - 전체 탭 → 카테고리 탭 → 즐겨찾기 탭 순서로 전환 플로우 검증
  - 카테고리 탭에서 즐겨찾기 저장 → 즐겨찾기 탭에서 확인 end-to-end 검증
  - 헤더 네비게이션을 통한 페이지 이동 후 탭 상태 초기화 여부 검증
  - 새로고침 후 즐겨찾기 상태는 유지되고 카테고리 탭은 전체로 초기화되는지 검증
